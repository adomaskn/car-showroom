# car-showroom

This repository holds a game-like car garage/showroom prototype.

## Run locally

1. Install deps: `npm install`
2. Start dev server: `npm run dev`

## Model location

Put your model at:

`public/models/cars/car.glb`

## Checkpoints

- Checkpoint 1: App scaffolding files exist and install succeeds.
- Checkpoint 2: 3D scene boots with floor and lights.
- Checkpoint 3: `car.glb` loads and appears centered on the floor.
- Checkpoint 4: Orbit controls and rendering loop run smoothly.
